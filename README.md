# Calculatrice Mobile

## Description
Calculatrice Mobile est une application simple et intuitive qui permet d'effectuer des calculs mathématiques de base et avancés. Que vous ayez besoin d'additionner, de soustraire, de multiplier, de diviser ou de résoudre des équations complexes, notre application est là pour vous aider.

## Fonctionnalités
- Addition, soustraction, multiplication et division
- Interface utilisateur conviviale et réactive


## Fonctionnalités à venir
- Calculs scientifiques (sin, cos, tan, log, etc.)
- Historique des calculs
- Support des thèmes clair et sombre

## Installation
1. Clonez ce dépôt sur votre machine locale :
    ```sh
    git clone https://github.com/AlphaSaiiko/calculatrice-mobile.git
    ```
2. Ouvrez le projet dans votre IDE préféré (par exemple, Android Studio).
3. Compilez et exécutez l'application sur un émulateur ou un appareil physique.

## Utilisation
1. Lancez l'application.
2. Utilisez les zones de textes pour entrer vos calculs.
3. Appuyez sur le bouton "+, - , / , x" pour obtenir le résultat.
